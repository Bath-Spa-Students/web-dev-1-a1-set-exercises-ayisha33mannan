âœŽ CSS

  • CSS stands for Cascading Style Sheets.

  • It is used to style the elements in a web page.

  • CSS is a styling language that works together with HTML to give a page its look and layout.

  • There is 3 ways to add CSS in HTML.

    1. Inline CSS :

      2. It is used directly in HTML elements using style attribute.

      Syntax:     <p style="color:blue"> I LOVE CODING </p>

      Result: I LOVE CODING

    2. Internal CSS :

      3. It is used in the <head> section of the HTML document within the <style> element.

      Syntax:    

      <head> 
       <style> 
       body{ 
       background-color: blueviolet; 
       } 
       </style> 
       </head>

    3. External CSS :

      4. It is used in a separate CSS file and linked to the HTML document using the <link> element in the <head> section.

      Syntax:    

      <head> 
       <link rel="stylesheet" href="name od the css file (with .css)">
      </head>

    Syntax:     body { background-color: rgb(255, 0, 0); }        (Here we have used color by using RGB)
